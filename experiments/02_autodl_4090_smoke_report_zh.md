# AutoDL RTX 4090 首次实测报告

日期：2026-07-08

这份报告记录 TinySeek-Lab 第一次远程 GPU 验证。所有 run 使用的是 toy byte-level 数据集，所以它们主要验证硬件、训练链路、显存和成本记录，不代表模型能力。

## 硬件环境

- 平台：AutoDL
- GPU：NVIDIA GeForce RTX 4090
- Driver：570.124.04
- CUDA runtime：12.8
- PyTorch：2.8.0+cu128
- Python：3.12.3
- 成本单价：2.18 元/小时

## 结论

- 这个 PyTorch 2.8.0 + CUDA 12.8 镜像可用。
- CUDA 环境下成本记录 JSON 能正常生成。
- 4090 足够跑当前 fp32 训练路径下的 115M dense 模型。
- 总参数 235M、激活参数约 84M 的 MoE 也能跑，峰值 allocated 显存约 6.67GB。
- 当前 toy 数据太小，只适合 smoke test；它会很快过拟合，不能拿来判断模型能力。
